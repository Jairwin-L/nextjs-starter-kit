# nextjs-blank-template

## Getting Started

First, run the development server:

```bash
vp install
vp run dev
```

Open [http://localhost:8060](http://localhost:8060) with your browser to see the result.

## Vite+ commands

```bash
vp run dev
vp run build
vp run verify
```
